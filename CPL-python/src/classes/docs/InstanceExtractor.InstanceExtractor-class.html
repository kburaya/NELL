learn(self, patternsPool, ontology, processedTextsPath)

source code 

Learning step of Instance extractor. Finds promoted instances for Categories in Ontology. Updates the promotedInstances dict of Category if found.

Parameters:
  • patternsPool (PatternsPool) - Exciting patterns pool
  • ontology (Ontology) - Exciting ontology to update
  • processedTextsPath (String) - path to processed texts directory
Returns: Ontology
updated Ontology with new instances for exciting categories or given Ontology if nothing found

evaluate(self, ontology, processedTextsPath, treshold=3)

source code 

Evaluating step of Instance Extractor. Chooses best promoted instances using precision score and adds them to Category in given Ontology. Precision: (num in text instance with Category / num in text).

Parameters:
  • ontology (Ontology) - Ontology to update
  • processedTextsPath (String) - path to processed texts folder
  • treshold (Integer) - number of instances to add by step
Returns: Ontology
updated Ontology with new instances if found or given one otherwise
