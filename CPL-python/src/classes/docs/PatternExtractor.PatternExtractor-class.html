learn(self, patternsPool, ontology, processedTextsPath)

source code 

Learning step of PatternExtractor. Finds promoted patterns in text using instances from ontology. Collects them into promotedPatternsPool. Calculates num of occurrence in text for promoted patterns.

Parameters:
  • patternsPool (PatternsPool) - Pool of exciting patterns
  • ontology (Ontology) - exciting ontology
  • processedTextsPath (String) - path to processed texts folder
Returns: Dictionary<String, Dictionary<String, Integer>>, PatternsPool
Dictionary of num of occurrence in text for patterns with categories from ontology, PatternsPool of found promoted patterns.

evaluate(self, ontology, patternsPool, promotedPatternsPool, promotedPatternsDict, processedTextsPath, treshold=3)

source code 

Evaluating step of PatternExtractor. Chooses best promoted patterns using precision score and adds them to exciting PatternsPool. Precision: (num in text pattern with Category / num in text).

Parameters:
  • ontology (Ontology) - exciting ontology
  • patternsPool (PatternsPool) - exciting PatternsPool to update
  • promotedPatternsPool (PatternsPool) - PatternsPool of promoted patterns
  • promotedPatternsDict (Dictionary<String, Dictionary<String, Integer>>) - Dictionary of num of occurrence in text for patterns with categories from ontology
  • treshold (Integer) - number of patterns to add by step
Returns: PatternsPool, Ontology(WHY??)
updated PatternsPool with new patterns if found.
